# GeoBrain — Setup Guide

## Your Files (put all 4 in one folder)
- index.html          the game
- GeoBrain-Admin.html admin panel  
- vercel.json         Vercel config (don't touch)
- README.md           this file

---

## STEP 1 — Upload to Vercel
1. Go to vercel.com and sign up free
2. Click "Add New" then "Project"
3. Drag your whole geobrain folder onto the page
4. Click Deploy
5. You get a URL like geobrain-abc123.vercel.app

---

## STEP 2 — Set Up Admin Panel
1. Go to your-url.vercel.app/admin
2. Password: admin12
3. Click the Global Sync tab
4. Your Bin ID is already filled in
5. Paste your NEW Master Key into the Master Key box
6. Click Save Credentials
7. Click Push Config to All Players
8. Click Test Connection — should say Connected!

---

## STEP 3 — Change Your Password
1. In admin panel go to Security tab
2. Current password: admin12
3. Set a new one
